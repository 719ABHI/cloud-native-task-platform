# Cloud Native Task Management Platform

A production-grade **Cloud Native Task Management Platform** built to demonstrate modern **Backend Development**, **Cloud**, and **DevOps** practices using Spring Boot, Docker, Kubernetes, GitHub Actions, Terraform, AWS, Helm, Prometheus, Grafana, Loki, Redis, and PostgreSQL.

---

# 🚀 Tech Stack

## Backend
- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSQL
